Frequently Asked Questions

How do I calculate AI API costs?
Multiply your token count by the per-token price. For example, 1M input tokens × $2.50/M = $2.50. Most APIs charge separately for input and output tokens. Our calculator does this automatically.
What is the cheapest AI API?
As of July 2026, the cheapest are: Gemini 2.0 Flash Lite ($0.075/M input), DeepSeek V4 Flash ($0.14/M input), and GPT-oss 20B ($0.08/M input). For mainstream models, GPT-4o mini ($0.15/M) is the most cost-effective.
How much does GPT-5 cost?
GPT-5 costs $1.25/M input tokens and $10.00/M output tokens. GPT-5 mini is cheaper at $0.25/M input and $2.00/M output. GPT-5.5 (flagship) costs $5.00/M input and $30.00/M output.
Should I use the cheapest model for everything?
No. Match the model to the task. Use cheap models (GPT-4o mini, DeepSeek) for chat and simple tasks. Use mid-tier (GPT-5 mini, Claude Sonnet) for complex work. Reserve flagship models (GPT-5.5, Claude Opus) for tasks requiring maximum capability.
