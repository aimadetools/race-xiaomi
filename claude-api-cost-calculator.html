How Claude API Pricing Works

Anthropic prices Claude models based on input tokens (your prompts) and output tokens (model responses). Pricing is per 1 million tokens. There are no monthly fees — you pay only for what you use.

Claude models are priced in tiers: Premium (Opus 4.7, 4 Opus) for complex reasoning, Mid-tier (Sonnet 4.6, Sonnet 4) for balanced performance, and Budget (Haiku 4.5) for high-volume simple tasks.

Claude API Pricing (2026)

  • Claude Opus 4.7: $5.00 input / $25.00 output per 1M tokens — 1M context, best for complex reasoning
  • Claude 4 Opus: $15.00 input / $75.00 output per 1M tokens — 200K context, legacy premium
  • Claude Sonnet 4.6: $3.00 input / $15.00 output per 1M tokens — 1M context, best value for most tasks
  • Claude Sonnet 4: $3.00 input / $15.00 output per 1M tokens — 200K context, previous gen
  • Claude Haiku 4.5: $1.00 input / $5.00 output per 1M tokens — 200K context, cheapest Claude

When to Use Each Claude Model

Claude Opus 4.7 — Use for complex multi-step reasoning, nuanced analysis, and tasks where quality is paramount. Worth the premium for legal, medical, or research applications.

Claude Sonnet 4.6 — The sweet spot for most developers. Excellent at code generation, content writing, RAG, and chatbots. 3x cheaper than Opus with 90% of the capability.

Claude Haiku 4.5 — Best for high-volume, simpler tasks: classification, routing, summarization, and chatbot responses. 5x cheaper than Opus.

How to Reduce Claude API Costs

  • Use Haiku for simple tasks: Route classification, routing, and simple chat to Haiku at $1/$5 — 5x cheaper than Opus
  • Optimize prompts: Shorter, more specific prompts reduce input tokens
  • Set max_tokens: Don't let Claude generate 4000 tokens when 200 will do
  • Use streaming: Streaming responses let users see results faster, reducing perceived latency
