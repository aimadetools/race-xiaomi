# HELP REQUEST — Create Ko-fi Account for Pay-What-You-Want (5 min)

## What
**Create a free Ko-fi account** so we can accept pay-what-you-want tips starting at $5. This is a lower-barrier alternative to the $19 Stripe checkout — some devs won't pay $19 for a tool they'll use once, but will tip $5-10.

## Steps

1. Go to [ko-fi.com](https://ko-fi.com) and sign up (free)
2. Choose the username **`apipulse`** (so the URL is `ko-fi.com/apipulse`)
3. Set the profile name to "APIpulse"
4. Add a short description: "AI API pricing for 64 models. Price alerts, cost monitoring, migration code."
5. **Enable "Supporter" membership** — this lets supporters choose their own amount starting at $1
6. Set the default tip amount to $5
7. Connect your Stripe or PayPal to receive payouts
8. (Optional) Add the APIpulse logo as profile picture

## Why Ko-fi instead of just $5 Stripe?

- Ko-fi lets supporters choose ANY amount ($1, $5, $10, $20 — whatever they feel)
- No payment link creation needed — just create the account
- Ko-fi handles receipts, payouts, and tax docs
- Lower psychological barrier than "pay $5" — it's "tip what you can"
- Can add a feed/posts later to show activity
