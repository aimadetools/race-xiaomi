🟢 OpenAI GPT-5.4 Family

OpenAI released four GPT-5.4 models in June 2026, filling the gap between GPT-5 mini and GPT-5.5. The family spans every price point — from the ultra-cheap nano to the premium Pro variant.

GPT-5.4

$2.50 input · $15.00 output
400K context · Mid tier · The new general-purpose workhorse

The direct successor to GPT-4o. Same $2.50 input price but 50% more output tokens and 3× the context window (400K vs 128K). Best for: chatbots, content generation, code assistance, and general reasoning tasks.

GPT-5.4 mini

$0.75 input · $4.50 output
400K context · Budget tier · Best value for most workloads

70% cheaper than GPT-5.4 with comparable quality for straightforward tasks. Ideal for: classification, summarization, data extraction, and high-volume API calls where cost matters more than peak intelligence.

GPT-5.4 nano

$0.20 input · $1.25 output
400K context · Ultra-budget · OpenAI's cheapest model ever

At $0.20 per million input tokens, GPT-5.4 nano is 92% cheaper than GPT-5.4. Still offers 400K context — unusual for a budget model. Best for: autocomplete, simple Q&A, routing, and embedding-like tasks at scale.

GPT-5.4 Pro

$30.00 input · $180.00 output
400K context · Premium tier · Maximum GPT-5.4 capability

The top-tier GPT-5.4 model. Same price as GPT-5.5 Pro but with 400K context (vs 1.05M). For complex multi-step reasoning, agentic coding, and tasks where output quality is paramount and budget is flexible.

🟣 Anthropic Claude Fable 5

Claude Fable 5 launched June 9, 2026 as Anthropic's most capable widely-released model. It slots above Opus 4.8 with a new tokenizer that produces ~30% more tokens per text compared to earlier Claude models.

Claude Fable 5

$10.00 input · $50.00 output
1M context · 128K max output · Premium tier · Adaptive thinking (always on)

Available on Claude API, Amazon Bedrock, Google Cloud, and Microsoft Foundry. Uses the new tokenizer from Opus 4.7 (~30% more tokens per text). Notable: no extended thinking support — uses adaptive thinking instead. Best for: complex reasoning, long-horizon agentic work, and tasks requiring maximum intelligence.

Claude Mythos 5 ⚠️ Restricted

$10.00 input · $50.00 output
1M context · 128K max output · Premium tier · Invitation-only via Project Glasswing

Anthropic's restricted model for cybersecurity and biology research. Same pricing as Fable 5 but access is limited to approved customers through Project Glasswing. Requires accepting a 30-day data retention policy for safety monitoring. The same model is offered as Claude Fable 5 with additional safeguards for public use.

Frequently Asked Questions

How much does GPT-5.4 cost?

GPT-5.4 costs $2.50 per 1M input tokens and $15.00 per 1M output tokens. It has a 400K token context window. The cheaper GPT-5.4 mini costs $0.75/$4.50 and the ultra-budget GPT-5.4 nano costs $0.20/$1.25.

What is Claude Fable 5 and how much does it cost?

Claude Fable 5 is Anthropic's most capable widely released model, launched June 9, 2026. It costs $10 per 1M input tokens and $50 per 1M output tokens with a 1M token context window and 128K max output. It uses adaptive thinking (always on) and the new tokenizer from Opus 4.7.

What is Claude Mythos 5?

Claude Mythos 5 is Anthropic's restricted-access model available through Project Glasswing. It costs $10/$50 per MTok (same as Fable 5) with a 1M context window. It's designed for cybersecurity and biology research. Access is invitation-only with a 30-day data retention policy. The same model is available as Claude Fable 5 with additional safeguards.

Which new model is cheapest?

GPT-5.4 nano is the cheapest new model at $0.20 per 1M input tokens and $1.25 per 1M output tokens. That's 84% cheaper than GPT-5.4 on input and 72% cheaper on output. For comparison, DeepSeek V4 Flash ($0.14/$0.28) is still cheaper on input but GPT-5.4 nano has a much larger 400K context window.

Is GPT-5.4 Pro worth the premium over GPT-5.5?

GPT-5.4 Pro costs $30/$180 per MTok — the same as GPT-5.5 Pro. Both have 400K+ context windows. If you need the absolute highest OpenAI capability, GPT-5.5 Pro ($30/$180, 1.05M context) offers more context. For most use cases, GPT-5.4 ($2.50/$15) provides excellent value at 12× less cost.
