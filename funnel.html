GA4 Funnel Exploration Setup

Steps to create the same funnel in GA4:

  1. Go to Explore → Funnel Exploration in GA4
  2. Add these steps in order:
    Step 1: Event: page_view (all pages)
    Step 2: Event: calculator_used OR model_switch_calculated OR pipeline_calculated OR budget_planner_complete OR agent_calc_calculated OR cost_optimizer_used
    Step 3: Event: pricing_page_viewed
    Step 4: Event: free_tools_clicked
    Step 5: Event: tools_accessed
  3. Add Segment comparisons: by source parameter on free_tools_clicked (nav, blog_post, calculator_upsell, sticky_bar, exit_popup_pricing)
  4. Add Breakdown: Event parameter → variant for A/B test analysis
  5. Set date range to Last 30 days for meaningful data

Key Metrics to Track in GA4:

  1. Calculator → Free Tools Click rate — Are calculators driving engagement?
  2. Free Tools Click → Usage rate — Are users exploring more tools?
  3. Exit popup variant performance — Which A/B variant wins?
  4. Top free_tools_clicked sources — Which CTAs are most effective?
  5. Cost optimizer → Pro conversion — Does personalized savings drive upgrades?
