# Help Request — Publish Chrome Extension to Web Store

**What:** Publish the APIpulse Chrome Extension to the Chrome Web Store so users can install it directly.

**Steps:**

1. **Create a Chrome Web Store developer account** (if you don't have one):
   - Go to https://chrome.google.com/webstore/devconsole
   - Pay the one-time $5 registration fee (deducted from APIpulse $100 budget)

2. **Package and upload the extension:**
   - The extension source is in `/home/race/race-xiaomi/chrome-extension/`
   - Zip the contents of the `chrome-extension/` folder (not the folder itself — the zip should contain manifest.json at the root)
   - In the Chrome Web Store Developer Dashboard, click "New Item"
   - Upload the zip file

3. **Fill in the store listing:**
   - **Name:** APIpulse — AI API Price Checker
   - **Description:** Free Chrome extension to check AI API pricing for 34 models across 10 providers. Instant cost estimates, model comparison, and budget calculator — right from your browser toolbar.
   - **Category:** Developer Tools
   - **Language:** English

4. **Upload screenshots:**
   - Take a screenshot of the extension popup (open chrome-extension/popup.html in a browser)
   - The landing page at getapipulse.com/chrome-extension.html has mockups you can reference

5. **Submit for review** — approval typically takes 1-3 business days

**Time:** 15-30 minutes

**Priority:** important — This is a distribution channel. The extension has a landing page already live, but users can't install it from the Chrome Web Store yet. Manual install is a barrier.

**Budget:** $5 (Chrome Web Store one-time registration fee)
