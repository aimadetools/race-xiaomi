Frequently Asked Questions

When was Claude 4 deprecated?

Claude 4 Opus and Claude Sonnet 4 were retired on June 15, 2026. API calls to these model IDs now return HTTP 410 Gone errors. You must migrate to a supported replacement.

What is the cheapest alternative to Claude 4 Opus?

DeepSeek V4 Flash is the cheapest at $0.14/$0.28 per million tokens — 99% cheaper than Claude 4 Opus ($15/$75). For similar quality, DeepSeek V4 Pro at $0.435/$0.87 is 97% cheaper. The direct Anthropic replacement, Claude Opus 4.8, is 67% cheaper at $5/$25.

Do I need a new API key to migrate from Claude 4?

No. If staying with Anthropic, your existing API key works with Claude Opus 4.8 and Sonnet 4.6. Only the model name string in your code changes. If switching providers, you'll need a new API key from the new provider.

How do I migrate from Claude 4 to another model?

Change one line of code: replace the model name string (e.g., 'claude-4-opus' → 'claude-opus-4-8'). Your API key, SDK, and endpoint stay the same. The entire migration takes under 5 minutes for most codebases.

It's after June 15 — what happens now?

All API calls to claude-4-opus and claude-sonnet-4 model IDs now return HTTP 410 Gone errors. Your application has stopped working until you update the model name. There is no grace period.

How much can I save by switching from Claude 4?

Savings range from 67% (Claude Opus 4.8) to 99% (DeepSeek V4 Flash). For a typical workload of 10M input + 2M output tokens/month, Claude 4 Opus costs $300/month. The same workload on Claude Opus 4.8 costs $100/month, and on DeepSeek V4 Pro it costs just $6.09/month.
