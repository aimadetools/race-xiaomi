APIpulse helps developers and technical founders make data-driven decisions about AI API costs. Instead of manually checking pricing pages across 10 different providers, APIpulse puts all the data in one place.

Features:
- Interactive cost calculator supporting 32 models across OpenAI, Anthropic, Google, Mistral, Cohere, Meta (Llama via Together.ai), and AI21
- Side-by-side model comparison tool with winner highlighting
- Sortable and filterable pricing index with all models
- Historical pricing trends (2023-2026) showing 90% average price drop
- 63 blog posts covering model comparisons, cost optimization strategies, and pricing analysis
- Calculator presets for startups, scale-ups, and enterprise usage
- Copy results to clipboard for sharing
- Dark/light theme, fully responsive, accessible

The Pro tier ($29 one-time) adds saved scenarios, PDF report export, and optimization recommendations.

Built as a static site — fast, no signup required, works instantly.
