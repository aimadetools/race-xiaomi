## What It Does
An embeddable cost calculator that lets users on ANY website calculate AI API costs. Select a model, enter tokens, see instant cost estimates. Links back to APIpulse for full comparison.

## Why Sites Would Embed It
- **Developer blogs** — readers can calculate costs inline while reading about AI APIs
- **Documentation sites** — add cost estimation to API docs
- **SaaS landing pages** — show prospects their potential AI costs
- **AI tool directories** — interactive element increases engagement

## Embed Code (copy-paste)
```html
<!-- Floating widget (bottom-right corner) -->
<script src="https://getapipulse.com/widget/calc-embed.js" defer></script>

<!-- OR: Inline embed -->
<div id="apipulse-calc" data-model="openai-gpt4o" data-tokens="100000"></div>
<script src="https://getapipulse.com/widget/calc-embed.js" defer></script>
```
