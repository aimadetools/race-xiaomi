# Backlink Building Strategy — APIpulse

**Goal:** 50+ referring domains by Week 8

---

## Phase 1: Quick Wins (Week 1-2) — ~15 backlinks

### Directory Submissions (9 backlinks)
- See: marketing/directory-submissions.md
- All content pre-written, human submits

### Existing Platforms (1 backlink)
- Product Hunt launch (content ready in marketing/product-hunt.md)

---

## Phase 2: Content Marketing (Week 2-4) — ~15 backlinks

### Dev.to Articles
- Cross-post 3-5 best blog posts to dev.to
- Each includes link back to getapipulse.com
- Target articles: "Cheapest LLM APIs", "Reduce AI API Costs", "API Pricing Cheat Sheet"
- **Format:** Slightly modified versions with dev.to-specific intro

### Hashnode / Medium
- Cross-post to Hashnode and/or Medium
- Canonical URL points to getapipulse.com
- Different audience than dev.to

### Guest Posts
- Pitch to: HackerNoon, Towards AI, The New Stack, InfoQ
- Angle: "We analyzed pricing across 7 LLM providers — here's what we found"
- Original data from calculator = unique content

### Stack Overflow / Reddit
- Answer questions about API pricing on Stack Overflow
- Helpful answers with link to calculator when relevant
- Reddit: contribute to r/MachineLearning, r/LocalLLaMA, r/artificial

## Phase 3: Outreach (Week 4-6) — ~10 backlinks

### Newsletter Mentions
- Pitch to developer newsletters:
  - TLDR AI
  - The Batch (Andrew Ng)
  - AI Tool Report
  - Ben's Bites
- Angle: "Free tool comparing 16 LLM models across 7 providers"

### Podcast Appearances
- Pitch to AI/developer podcasts
- Angle: "Built a micro-SaaS for API cost comparison in 12 weeks"
- Show HN story + technical approach

### Tool Roundup Articles
- Monitor for "best AI tools" / "developer tools" roundup articles
- Reach out to authors with APIpulse inclusion pitch
- Google Alerts for: "AI API tools", "LLM pricing tools"

---

## Phase 4: Community (Week 6-8) — ~10 backlinks

### Open Source Contribution
- Create GitHub repo with pricing data as JSON
- Community can contribute/verify pricing updates
- Links back to site for the full calculator experience

### Integration Partners
- Reach out to AI frameworks (LangChain, LlamaIndex, Haystack)
- Propose APIpulse as recommended cost estimation tool
- Blog post about integration

### Comparison Sites
- Submit to AI comparison/review sites
- "AI tools directory" searches yield many directories
- Each submission = backlink

## Content Angles for Outreach

1. **Data-driven:** "We compared pricing across 7 LLM providers — here's the data"
2. **Cost-saving:** "How to cut your AI API bill by 40% (with proof)"
3. **Builder story:** "I built a micro-SaaS for API cost comparison"
4. **Trend piece:** "LLM prices dropped 90% in 3 years — what's next?"
5. **Tool recommendation:** "Free tool for estimating AI API costs"

---

## Key Principle

Every backlink effort should also drive direct traffic. Directory submissions, guest posts, and newsletter mentions all bring visitors — the SEO benefit is a bonus.
