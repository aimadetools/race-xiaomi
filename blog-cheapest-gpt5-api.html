The Four Budget Tiers

Ultra-Budget: GPT-oss 20B ($0.08/$0.35)

OpenAI's open-weight model is the absolute cheapest GPT-5 option. At $0.08 per million input tokens, it costs less than a penny per request. Best for high-volume, simple tasks: classification, extraction, sentiment analysis, chatbot routing.

Budget: GPT-oss 120B / GPT-4o mini ($0.15/$0.60)

Both priced identically, these are workhorses for everyday tasks. GPT-oss 120B runs on open weights (self-host or via providers like Together.ai), while GPT-4o mini is a hosted API with built-in safety. Good for summarization, Q&A, content generation.

Mid-Range: GPT-5 mini ($0.25/$2.00)

The sweet spot for most developers. GPT-5 mini offers near-GPT-5 quality at a fraction of the cost. With 272K context, it handles complex multi-turn conversations and code generation. 67% cheaper than GPT-5.

Premium: GPT-5 ($1.25/$10.00)

Full GPT-5 for when you need maximum reasoning. Reserve this for the 10-20% of requests that genuinely require it — not every API call.

The Bottom Line

For most use cases, GPT-5 mini ($0.25/$2.00) is the cheapest GPT-5 API worth using. It delivers 80%+ of GPT-5's quality at 80% lower cost. For truly budget-conscious high-volume workloads, GPT-oss 20B ($0.08) is hard to beat — and if you need even cheaper, Gemini Flash Lite edges it out on input cost.
